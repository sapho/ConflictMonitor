# Image Manipulation for Change Detection

### Required Folderstructure
```bash
├── ImageManipulation
│   ├── tifs (exactly 2 .tifs inside)
│   │   ├── *tif1.tif
│   │   ├── *tif2.tif
│   ├── script.py
│   ├── RESULT.tif (will be generated where script.py is located)
```

### Usage
```sh
$ cd ImageManipulation
$ python3 script.py
```
